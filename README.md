# Mutual Fund Portfolio Calculator

This project fetches NAV data from AMFI and allows users to:

- Select a mutual fund
- Enter number of units
- Calculate portfolio value instantly

## Data Source
https://www.amfiindia.com/spages/NAVAll.txt

## Features
- Live NAV fetching
- Simple UI
- No backend required

## How to Run
1. Clone repo
2. Open `index.html` in browser

## Note
Uses a CORS proxy (allorigins) to fetch AMFI data.
